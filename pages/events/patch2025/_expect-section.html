Here's What to Expect

We cannot wait to introduce you to fullstack web development. In the two-hour workshop, we hope that we can provide the following experiences:

  • Be introduced to fullstack web development, and the client-server model.
  • Learn how to build a basic API with Django.
  • Familiarize yourself with API testing using Postman.
  • Get acquainted with React and integrating APIs into your web app.
  • Discover how Docker eases development and deployment.
