Code of Conduct

Dev8 PATCH 2025: Fullstack Web 101 is organized to provide a safe space for knowledge sharing, and forming new connections. We want all participants to have an enjoyable experience in our events. As such, all attendees, speakers, sponsors, organizers, volunteers, and staff of any Dev8 events are required to comply with the Dev8 Event Code of Conduct, which requires everyone to be respectful to each other, observant of venue rules, and welcoming to everyone. We will enforce this code throughout Dev8 PATCH 2025: Fullstack Web 101.

If you believe someone is violating the Dev8 Event Code of Conduct, or have any other concerns, please do not hesitate to inform us via dev8community@gmail.com.
