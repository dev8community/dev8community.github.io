New Year, New Start

A new year means another opportunity for growth, learning, and, most importantly, starting up with something new. Inspired from the technical term of “booting up” a computer, Dev8 is introducing its newest event, Dev8 BOOT 2025, a virtual conference for Eastern Visayas that aims to provide a platform for knowledge sharing and inspiring developers, students, and tech enthusiasts to do amazing things in 2025.

Code of Conduct

Dev8 BOOT 2025 is organized to provide a safe space for knowledge sharing, and forming new connections. We want all participants to have an enjoyable experience in our events. As such, all attendees, speakers, sponsors, organizers, volunteers, and staff of any Dev8 events are required to comply with the Dev8 Event Code of Conduct, which requires everyone to be respectful to each other, observant of venue rules, and welcoming to everyone. We will enforce this code throughout the Dev8 BOOT 2025.

If you believe someone is violating the Dev8 Event Code of Conduct, or have any other concerns, please do not hesitate to inform us via dev8community@gmail.com.
